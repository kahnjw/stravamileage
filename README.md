# Strava Mileage

Record mileage on your gear. Know when to replace it.

## Get (git) It

```
$ git clone git@github.com:kahnjw/stravamileage.git
$ cd stravamileage
```

## Set It Up

```
$ npm install
$ bower install
```

## Test It

```
$ grunt test
```

Or if you prefer to use an in browser test runner do:

```
$ testem
```

## Build It

```
$ grunt
```
